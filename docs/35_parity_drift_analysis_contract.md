# Parity Drift Analysis Contract

## Goal

Quantify parity-metric drift across scenario families from replay reports.

## Input

Replay report JSON files generated by measured replay execution:

- `<run>/measured_replay_outputs.../pack_*/replay_report.json`

## CLI

```bash
PYTHONPATH=src python3 /Users/seongcheoljeong/Documents/Codex_test/scripts/analyze_parity_drift_from_replay_reports.py \
  --report baseline=/path/to/baseline_replay_report.json \
  --report candidate=/path/to/candidate_replay_report.json \
  --quantiles 0.5,0.9,0.99 \
  --output-json /path/to/parity_drift_report.json
```

The first `--report` is treated as baseline.

## Output JSON

- `baseline`
- `scenarios[]`
  - `candidate_count`, `pass_rate`
  - `metric_quantiles`
- `drift_vs_baseline[]`
  - `metric_drift[metric][quantile]`
    - `baseline`, `candidate`, `delta`, `ratio`
