# Classic Dashboard 문서 맵

## 목적

classic dashboard 문서를 써야 하는 것은 알지만, 어느 문서부터 열어야 할지 모를 때 이 페이지를 사용합니다.

이 페이지는 classic dashboard 문서 세트의 routing map입니다.
